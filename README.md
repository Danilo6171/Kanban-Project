# 📋 Kanban Task Board

Um gerenciador de tarefas dinâmico baseado na metodologia Kanban, desenvolvido para organizar fluxos de trabalho que faço no meu dia a dia em três etapas: A Fazer, Em Andamento e Concluído. 

Este projeto foi construído puramente com tecnologias web nativas, sem dependências externas, utilizando armazenamento local para persistência de dados.

---

## 🚀 Funcionalidades

- **Criar Tarefas**: Adicione títulos, descrições detalhadas e níveis de prioridade.
- **Categorização por Prioridade**: Cores visuais dinâmicas para prioridades Baixa, Média e Alta.
- **Movimentação entre Colunas**: Fluxo de trabalho interativo através de botões de navegação lateral.
- **Persistência de Dados**: Integração com `localStorage` para manter os dados salvos mesmo após fechar o navegador.
- **Exclusão de Itens**: Remoção simples e rápida de cards concluídos ou cancelados.
- **Interface Responsiva**: Layout adaptável para dispositivos móveis, tablets e desktops.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza as tecnologias web fundamentais (Vanilla Tech Stack):

- **HTML5**: Estruturação semântica do quadro, colunas e formulários em janelas modais.
- **CSS3**: Estilização moderna utilizando CSS Grid, Flexbox, variáveis nativas (`:root`) e efeitos de transição.
- **JavaScript (ES6+)**: Manipulação assíncrona do DOM, gerenciamento de estado e persistência de dados.
